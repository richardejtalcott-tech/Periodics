# Periodic v2 update

This patch upgrades the tested Periodic foundation with:

- all 118 element names
- search by name, symbol, or atomic number
- center/reset control
- element names shown while zoomed in
- dimensional tile gradients
- richer element detail pages
- draggable atom rotation
- pinch-to-zoom atom view
- v2 build metadata and GitHub Actions artifact

Scientific note: atomic mass and neutron values in this early visual build are labeled approximate. A later database module should replace them with curated isotope/property data.
