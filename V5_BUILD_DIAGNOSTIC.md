# Periodic v5 build diagnostic

This branch is not approved until the PR workflow produces:

1. A successful React Native Android debug build.
2. An APK larger than 5 MB.
3. Emulator screenshots of the splash, periodic table, and element explorer.
4. No old v4 magnifying-glass layout or permanent table tilt.

Failed runs must be diagnosed from the exact GitHub Actions job logs before another APK is distributed.
